##  Vim window

Vertical split
```sh
:vsp file1.txt
```

Open another file on the right 
```sh
:sp file2.txt
```

Open to new tab
```sh
:tabe file3.txt
```

## Seesion

Create session
```sh
:mks
```

Close out of vim
```sh
:wqa
```

make ls to the current directory and observe the Session.vim file

Restore session
```sh
:vim -S
```
